# Singularity 🔐

**Singularity** is a lightweight and secure password manager for Android, built with Kotlin and modern development practices. Focus on privacy: no cloud, no tracking, just your data under your control.

## ✨ Features
* **Offline-First:** Your data never leaves your device. Powered by **Room Database**.
* **Military-Grade Encryption:** Sensitive information is protected using **AES-256** via `Jetpack Security (EncryptedSharedPreferences)`.
* **Modern UI:** Built entirely with **Jetpack Compose** and Material 3 design principles.
* **Biometric Ready:** Architected to support secure local authentication.

## 🛠 Tech Stack
* **Language:** [Kotlin](https://kotlinlang.org/)
* **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
* **Local Storage:** [Room Persistence Library](https://developer.android.com/training/data-storage/room)
* **Architecture:** MVVM (ViewModel, StateFlow)
* **Security:** [Jetpack Security Crypto](https://developer.android.com/jetpack/androidx/releases/security)
